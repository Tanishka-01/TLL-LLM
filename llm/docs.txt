# Run the setup script to install dependencies and environment
```bash
./setup.sh
```

# after install check if working
```bash
ollama list
```

# install model or run script to install one from hugging face
## either this:
```bash
ollama pull deepseek-r1:1.5b
```
## or this:
```bash
./create-ollama-model [url]
```
